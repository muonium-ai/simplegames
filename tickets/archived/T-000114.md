## Progress Log
- 2026-05-22: Added time.monotonic() per-level start tracking and terminal completion prints in all 6 Tier-1 multi-level games. bricks: records level_start_time after intro/advance overlay; prints '[bricks] Level N (NAME) completed in X.XXs' on every level clear and resets for the next. flappybird: level_start_time set in reset_game and advance_level; prints '[flappybird] Level N completed in X.XXs' (cleared_level captured before apply_level). tetris: level_start_time set at game start and on R-restart; prints '[tetris] Level N completed in X.XXs' on step-up (lines_cleared_total >= level*10) before bumping level, then resets timer. 2048: Game.start_time captured in __init__; prints '[2048] Milestone <N> reached in X.XXs' for each newly_unlocked milestone (using elapsed since game start). minesweeper: level_start_time set in reset_game; prints '[minesweeper] Difficulty <name> cleared in X.XXs' in handle_victory (added idempotent guard to avoid double-fire). sudoku: level_start_time recorded at start of play_round; prints '[sudoku] Difficulty <name> solved in X.XXs' when grid.is_solved(). All 6 files pass python3 -m py_compile. All flush=True. No UI changes; existing ESC/save/autoplay/solver behavior preserved.
