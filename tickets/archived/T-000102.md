## Progress Log
- 2026-05-22: Implemented greedy autoplay bot in tetris/tetris.py. Added --autoplay argparse flag (with --level honored) that skips the start screen. Bot enumerates every (rotation, column) placement for the active piece, simulates the locked board (drop + row clear), and scores with weights +1*lines_cleared, -0.5*aggregate_heights, -0.7*holes, -0.2*bumpiness. Plan is computed once per active piece; piece advances toward target one rotation/column per ~60ms tick so it remains watchable; gravity untouched. HUD adds '[AI]' next to Level when autoplay is on. Preserved ESC quit, non-blocking game-over deadline, level-select / --level argv path, level-up overlay, and pause/restart keys. py_compile passes; headless smoke test with SDL_VIDEODRIVER=dummy runs cleanly.
